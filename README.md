# Установка
* Установить игровой движок [LÖVE 2D](https://love2d.org/)
* Скачать проект
* Сделать ```.zip``` архив из папки ```src``` (Без папок, только файлы)
* Поменять расширение с ```.zip``` на ```.love```
* Запустить

## Как должен выглядеть ```.zip``` архив
* ai.lua
* board.lua
* conf.lua
* gui.lua
* main.lua
* utils.lua
